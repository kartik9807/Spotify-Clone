# Spotify-Clone
Spotify-inspired music player web app built using HTML, CSS, and JavaScript.
Features dynamic album loading, audio playback controls (play/pause, next/previous), responsive UI, and real-time state management.
Focused on understanding core frontend concepts like async/await, fetch API, event handling, and flexbox-based layouts.
